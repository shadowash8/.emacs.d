* Overview
This repository contains my personal Emacs configuration.  

The repo lives at:
#+begin_src shell
~/.emacs.d
#+end_src

* Features
- Modular configuration (split across =lisp/= and =lisp/packages/=).
- Package management with [[https://github.com/jwiegley/use-package][use-package]].
- Custom evil keybindings, Org-mode setup.
- Optimized for performance.
- Theming and UI tweaks for a clean editing experience.
- Mostly usable lsp support.
- Automatic theme sync with xresources.

* Installation
Clone the repo into your home directory:
#+begin_src shell
git clone https://codeberg.org/shadowash8/.emacs.d.git ~/.emacs.d
#+end_src

* File Structure
#+begin_src text
.emacs.d/
├── init.el          ;; Entry point
└── lisp/            ;; Modular config
    ├── config.el
    ├── functions.el
    ├── keymaps.el
    ├── packages.el
    └── packages/
        ├── editing.el
        ├── lisp.el
        ├── org.el
        ├── themes.el
        └── ui.el
#+end_src

* Usage
Start Emacs normally:
#+begin_src shell
emacs
#+end_src

Or use the daemon + client setup:
#+begin_src shell
emacs --daemon
emacsclient -c
#+end_src
